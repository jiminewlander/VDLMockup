## What is IRENE-AE9/AP9/SPM?

IRENE (International Radiation Environment Near Earth) is a suite of models for the fluxes of radiation belt and plasma particles in near-Earth space. The suite includes:

| Model | Description |
|-------|-------------|
| **AE9** | Energetic Electrons |
| **AP9** | Energetic Protons |
| **SPM** | Space Plasma Model |

## Key Features

- Derived from **45 satellite data sets** measured by on-board sensors
- Provides **statistical confidence intervals** (e.g. median and 95th percentile)
- Supports **space system design**, mission planning, and climatological specification
- Available as **Windows executable** and **source code** (by request)
- **C, C++, and Python APIs** for programmatic access

## Supported Platforms

- Windows (32-bit and 64-bit) — distributed as executable
- Linux / other platforms — source code available by request (contact team)

## Current Version

**V1.58.001** (released 4 March 2024)

## Applications

- Radiation environment specification for spacecraft design
- Mission planning for orbital environments
- Statistical analysis of radiation belt variability
- Dose calculation and effects estimation

For the full technical overview, see: G. P. Ginet, et al., "AE9, AP9 and SPM: New models for specifying the trapped energetic particle and space plasma environment," *Space Sci. Rev.*, 179, 579-615 (2013). Available on the [Documents](documents) page.
