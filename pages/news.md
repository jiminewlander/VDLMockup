# IRENE-AE9/AP9/SPM News

This page provides announcements of new releases, bug reports, and other information of interest to users. For questions, contact the AE9/AP9/SPM Team at [AFRL.RVBXR.AE9.AP9.Org.Mbx@us.af.mil](mailto:AFRL.RVBXR.AE9.AP9.Org.Mbx@us.af.mil).

---

- **4 March 2024: V1.58.001 release.** This version adds a feature to generate variable timestep ephemeris, useful for elliptical orbits. It also simplifies specification of input database parameters, supports output of local magnetic field vector components with the 'Adiabatic' output option, and provides other API improvements. See the [V1.58.001 release notes](../assets/docs/IRENE_V1_58_001_ReleaseNotes.pdf) for more information.

- **1-2 March and 15 May 2023: IRENE Industry Days.** Following the public release of AE9/AP9 v1.57, the model development team provided an overview of recent feature updates and a preview of expected changes in upcoming versions. The presentations are available on the [Documents page](documents) under "Presentations".

- **11 July 2022: V1.57.004 release.** This version includes a kernel-based capability for dose calculations, replicating dose vs. depth results from SHIELDOSE2 but enabling faster calculations. It also includes new features for accumulators and an expanded C and Python API. See the [V1.57.004 release notes](../assets/docs/IRENE_V1_57_004_ReleaseNotes.pdf) for more information.

- **1 December 2017: V1.50.001 release.** This version updates the AP9 and AE9 flux maps with data from the Van Allen Probes mission. It also adds proton data from HiLET and Azur, the first international data set. See the [V1.50.001 release notes](../assets/docs/Ae9Ap9_v1_50_001_ReleaseNotes.pdf).

- **23 January 2017: V1.35.001 release.** This version introduces support of parallelized processing for Linux and 64-bit Windows platforms. See the [V1.35.001 release notes](../assets/docs/Ae9Ap9_v1_35_001_ReleaseNotes.pdf).

- **9 February 2016: V1.30.001 release.** This version addresses the Monte Carlo issue affecting AP9 and AE9. See the [V1.30.001 release notes](../assets/docs/Ae9Ap9_v1_30_001_ReleaseNotes.pdf).

- **14 October 2015: V1.20.004 release.** This version fixes a software bug in the "IntegralPlasma" utility. See the [V1.20.004 release notes](../assets/docs/Ae9Ap9_v1_20_004_ReleaseNotes.pdf).

- **13 March 2015: V1.20.002 release.** First public release since V1.05.001, incorporating updated flux maps for AE9, AP9 and SPM electrons and protons, plus several feature improvements. See the [V1.20.002 release notes](../assets/docs/Ae9Ap9_v1_20_002_ReleaseNotes.pdf).
